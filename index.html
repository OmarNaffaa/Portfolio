5G Self-Healing Network Simulator
Undergraduate Research: 2018 - 2020

This project aims to simulate a network comprised of current technologies in order to test various self-healing and network management strategies in the hopes of implementing a digital network manager capable of detecting the “unhealthy/failing” cells present in a network to implement researched self-healing procedures. The simulator will accurately model channel conditions and model downlink communications between cellular towers and User Equipment (cell phones, computers, etc.). The project centers around determining the optimal algorithms implemented at the management level for achieving this result using the constraints introduced by the environment and technologies simulated.

In 2020, this project won best project within the Electrical and Computer Engineering department. A project overview video can be viewed here.

Decentralized Renewable Wastewater Treatment System (DROWT)
Undergraduate Research: 2018 - 2019

This project presents the development of a mock control system for an automated, self-sustaining, decentralized water treatment plant. The system is composed of an Arduino microcontroller MKR1000 and an array of sensors linked to a mobile application to monitor different aspects of the system. The application lets the end user monitor the entire system and grants access to the control of certain sensors. The system communicates with cloud based analytic software ThingSpeak to process and display system data on a mobile android application.

This project was presented at COMPSAC 2019 and published in an IEEE Peer-Reviewed journal.
